FAQs
----

### Is a Google Docs contract legally binding?



Yes, a Google Docs contract can be legally binding if it meets the requirements of a valid contract, including mutual agreement and consent by all parties. E-signatures can be used to finalize the contract.

### Can I add signatures to a Google Docs contract?



While Google Docs doesn’t have a built-in signature tool, you can insert signatures using third-party e-signature tools like DocuSign or HelloSign, or by manually adding an image of your signature.

### How do I share a Google Docs contract?



You can share a Google Docs contract by clicking the “Share” button in the top-right corner. You can send it via email, generate a link, or set specific permissions for collaborators or viewers.

### How secure is a Google Docs contract?



Google Docs offers strong security features, including encryption and two-factor authentication. However, it's important to set appropriate access permissions and be cautious when sharing sensitive information.

### Can I track changes in a Google Docs contract?



Yes, Google Docs offers a version history feature, allowing you to see all changes made to the contract and revert to previous versions if needed. This ensures transparency and keeps track of edits.
