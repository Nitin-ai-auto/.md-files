Why ScreenFlow Shortcuts Matter
-------------------------------

Using shortcuts can dramatically improve your editing speed and workflow efficiency. Instead of spending time moving your mouse to find the right tool or command, you can perform tasks instantly with a simple keystroke. This not only saves time but also keeps you immersed in your creative flow without interruptions. Whether you’re a beginner or a seasoned expert, mastering these shortcuts can make a significant difference in your editing experience.

Essential Keyboard Shortcuts
----------------------------

### General Shortcuts

*   Command + N: Create a new document
*   Command + O: Open an existing document
*   Command + S: Save your project
*   Command + Z: Undo the last action
*   Command + Shift + Z: Redo the last undone action

### Editing Shortcuts

*   Command + C: Copy selected clips
*   Command + V: Paste clips from the clipboard
*   Command + X: Cut selected clips
*   Delete: Remove selected clips from the timeline
*   Command + B: Split the selected clip at the playhead

### Playback Shortcuts

*   Space: Play or pause the video
*   Left Arrow: Move the playhead back 1 second
*   Right Arrow: Move the playhead forward 1 second
*   L: Play forward at double speed
*   J: Play backward at double speed

### Screen Recording Shortcuts

*   Command + Shift + 2: Start a new screen recording
*   Command + Shift + 3: Take a screenshot of the entire screen
*   Command + Shift + 4: Take a screenshot of a selected area

Customizing Shortcuts
---------------------

ScreenFlow allows you to customize shortcuts according to your preferences. To set custom shortcuts, navigate to `ScreenFlow > Keyboard Shortcuts` in the menu bar. Here, you can modify existing shortcuts or create new ones that enhance your workflow according to your editing style.

Tips for Remembering Shortcuts
------------------------------

*   Create a Cheat Sheet: Write down the most used shortcuts and place it by your workspace.
*   Frequent Practice: Regularly practice using shortcuts instead of the mouse until it becomes second nature.
*   Use Mnemonics: Associate a shortcut with a phrase or image that helps you remember it better.
