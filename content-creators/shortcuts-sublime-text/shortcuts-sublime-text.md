Getting Started: Basic Shortcuts
--------------------------------

Understanding the essentials is the first step in harnessing the power of Sublime Text. Here are some foundational shortcuts every new user should memorize:

*   New File: `Ctrl + N` (Windows/Linux), `Cmd + N` (Mac)
*   Open File: `Ctrl + O` (Windows/Linux), `Cmd + O` (Mac)
*   Save File: `Ctrl + S` (Windows/Linux), `Cmd + S` (Mac)
*   Close File: `Ctrl + W` (Windows/Linux), `Cmd + W` (Mac)

These shortcuts will streamline your workflow right from the start.

Navigating Your Code Efficiently
--------------------------------

Navigating efficiently within your code can drastically reduce the amount of time spent searching for files and lines. Here’s how:

*   Goto Anything: `Ctrl + P` (Windows/Linux), `Cmd + P` (Mac) – Quickly access files, symbols, or lines.
*   Show Code Navigation: `Ctrl + G` (Windows/Linux), `Cmd + G` (Mac) – Jump directly to a specific line number.
*   Show/Hide Side Bar: `Ctrl + K, Ctrl + (1-9)` (Windows/Linux/Mac) – Toggle visibility of your project sidebar.

Familiarizing yourself with these shortcuts will help you navigate large codebases more effectively.

Editing Text Like a Pro
-----------------------

Editing is just as crucial as navigation. Here are some shortcuts to enhance your text editing skills:

*   Select All: `Ctrl + A` (Windows/Linux), `Cmd + A` (Mac)
*   Duplicate Line: `Ctrl + D` (Windows/Linux), `Cmd + D` (Mac) – Perfect for quickly creating copies of code.
*   Comment/Uncomment Line: `Ctrl + /` (Windows/Linux), `Cmd + /` (Mac) – Make debugging easier by toggling comments.

Utilizing different editing shortcuts can help in optimizing your coding processes, allowing for rapid adjustments.

Advanced Shortcuts for Power Users

For those who are ready to elevate their skills, these advanced shortcuts are game-changers:

*   Multiple Selections: `Ctrl + Click` or `Cmd + Click` (Mac) – Edit multiple lines or selections simultaneously.
*   Replace All: `Ctrl + H` (Windows/Linux), `Cmd + Alt + F` (Mac) – Perform search and replace actions across your entire document.
*   Open Console: `Ctrl + `` (Windows/Linux), `Cmd + `` (Mac) – Access the console for debugging and inputting commands.

Mastering these shortcuts allows for a more dynamic and effective coding environment.

Creating Custom Shortcuts
-------------------------

Sublime Text offers the ability for users to customize shortcuts to better suit their workflows. Here’s how to set it up:

1.   Go to `Preferences` -> `Key Bindings`.
2.   In the User Key Bindings file, you can define your custom shortcuts in JSON format. For example:

    [ { "keys": ["ctrl+shift+n"], "command": "new_window" }
    ] 
1.   Save the changes and start using your personalized shortcuts immediately.

Creating custom shortcuts can streamline repetitive tasks and enhance your coding efficiency.
