Essential Sheets Shortcuts
--------------------------

### Navigation Shortcuts

*   Arrow Keys: Move one cell in the direction of the arrow.
*   Ctrl + Arrow Key (Windows) / Command + Arrow Key (Mac): Jump to the edge of the data region.
*   Home: Move to the beginning of the row.
*   Ctrl + Home: Jump to the top left of the spreadsheet.

### Selection Shortcuts

*   Shift + Arrow Keys: Select cells while navigating.
*   Ctrl + A: Select all cells in the spreadsheet.
*   Ctrl + Shift + Space: Select the entire sheet.

### Formatting Shortcuts

*   Ctrl + B: Toggle bold formatting.
*   Ctrl + I: Toggle italic formatting.
*   Ctrl + U: Toggle underline formatting.
*   Ctrl + 1: Open the Format Cells dialog box for more customization.

### Editing Shortcuts

*   Enter: Confirm data entry and move down to the next cell.
*   Esc: Cancel any current edits.
*   Ctrl + Z: Undo the last action.
*   Ctrl + Y: Repeat the last action.

### Formula Shortcuts

*   = (equals sign): Start a formula.
*   Tab: Autocomplete the function name as you start typing.
*   Ctrl + Shift + Enter: Create an array formula.

### Collaboration Shortcuts

*   Ctrl + K: Insert a hyperlink in a cell.
*   Ctrl + P: Print the current sheet.
*   Ctrl + Alt + Shift + H: Open the comment section to add or view comments.

Frequently Asked Questions (FAQs)
---------------------------------

### What are keyboard shortcuts in Google Sheets?

Keyboard shortcuts are combinations of keys that perform specific commands in Google Sheets, allowing for quicker navigation and data manipulation.

### How do I customize my shortcuts in Google Sheets?

While you cannot modify existing keyboard shortcuts in Google Sheets, you can use Google’s built-in keyboard shortcuts alongside custom Google Apps Scripts for more advanced functionalities.

### Are there any shortcuts that work on both Windows and Mac?

Yes, many shortcuts, like selecting (Ctrl + A or Command + A), work on both Windows and Mac operating systems with slight variations in key usage.
