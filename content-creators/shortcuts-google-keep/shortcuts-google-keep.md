Google Keep Keyboard Shortcuts
------------------------------

Below is a list of essential Google Keep keyboard shortcuts designed to enhance your user experience:

### Creating and Managing Notes

*   New Note: Press Ctrl + N (Windows) or Cmd + N (Mac) to create a new note quickly.
*   Save Note: Use Ctrl + S (Windows) or Cmd + S (Mac) to save the current note you are working on.
*   Delete Note: To delete a note, press Ctrl + Backspace (Windows) or Cmd + Backspace (Mac).

### Navigation Shortcuts

*   Open Search: Quickly access the search feature by hitting Ctrl + F (Windows) or Cmd + F (Mac).
*   Go to List View: Press G followed by L to switch to your list view.
*   Open Menu: Use Ctrl + Shift + M (Windows) or Cmd + Shift + M (Mac) to open the main menu.

### Formatting Shortcuts

*   Bold Text: Highlight text and press Ctrl + B (Windows) or Cmd + B (Mac) for bold formatting.
*   Italicize Text: Use Ctrl + I (Windows) or Cmd + I (Mac) to italicize selected text.
*   Create a Checklist: Easily create to-do lists by pressing Ctrl + Shift + 8 (Windows) or Cmd + Shift + 8 (Mac).
