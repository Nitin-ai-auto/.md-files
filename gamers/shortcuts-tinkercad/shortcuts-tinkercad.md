Essential Tinkercad Shortcuts
-----------------------------

Let’s go over the most useful shortcuts that every Tinkercad user should know.

### 1. Basic Navigation Shortcuts

*   **Zoom In/Out:**`Ctrl + Mouse Scroll` — Use this to easily zoom in on small areas or zoom out for a broader view of your design.
*   **Pan View:**`Shift + Mouse Drag` — Move your entire view without altering the design.

### 2. Selection Shortcuts

*   **Select All:**`Ctrl + A` — Quickly selects all objects within your workspace.
*   **Deselect:**`Esc` — Deselects the currently selected object.

### 3. Transformation Shortcuts

*   **Copy:**`Ctrl + C` — This classic shortcut allows for object replication.
*   **Paste:**`Ctrl + V` — Place your copied object quickly.
*   **Duplicate:**`Ctrl + D` — Quickly creates a copy of the selected object and places it over the original.

### 4. Object Manipulation Shortcuts

*   **Delete:**`Delete` — Remove selected items effortlessly.
*   **Group:**`Ctrl + G` — Merge multiple objects into a group for easier manipulation.
*   **Ungroup:**`Ctrl + Shift + G` — Breaks apart grouped objects for individual adjustments.

### 5. Viewing Options

*   **Toggle Grid Snap:**`Ctrl + Shift + G` — Switch between snapping to the grid and free moving.
*   **Change View Orientation:**`Shift + One of the Number Keys (1-9)` — Quickly switch between predefined views to better inspect your design.
