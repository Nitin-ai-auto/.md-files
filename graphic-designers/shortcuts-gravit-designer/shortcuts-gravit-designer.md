Why Use Keyboard Shortcuts?
---------------------------

Keyboard shortcuts can dramatically improve your efficiency in Gravit Designer. They save time by allowing you to perform frequent tasks without navigating through menus. This means fewer interruptions in your creative flow and more focus on your design projects.

Fundamental Gravit Designer Shortcuts
-------------------------------------

### Basic Navigation Shortcuts

*   Zoom In / Zoom Out: 
    *   **Windows:** Ctrl + + / Ctrl + -
    *   **Mac:** Cmd + + / Cmd + -

*   Fit to Screen: 
    *   **Windows:** Ctrl + 0
    *   **Mac:** Cmd + 0

*   Pan (Hand Tool): 
    *   **Windows/Mac:** Hold Spacebar while dragging the canvas.

### Selection and Editing Shortcuts

*   Select All: 
    *   **Windows:** Ctrl + A
    *   **Mac:** Cmd + A

*   Deselect: 
    *   **Windows:** Ctrl + Shift + A
    *   **Mac:** Cmd + Shift + A

*   Undo / Redo: 
    *   **Windows:** Ctrl + Z / Ctrl + Shift + Z
    *   **Mac:** Cmd + Z / Cmd + Shift + Z

### Shape & Object Shortcuts

*   Rectangle Tool: 
    *   **Windows/Mac:** Press R

*   Ellipse Tool: 
    *   **Windows/Mac:** Press E

*   Pen Tool: 
    *   **Windows/Mac:** Press P

*   Group Objects: 
    *   **Windows:** Ctrl + G
    *   **Mac:** Cmd + G

### Text Shortcuts

*   Text Tool: 
    *   **Windows/Mac:** Press T

*   Bold Text: 
    *   **Windows:** Ctrl + B
    *   **Mac:** Cmd + B

Advanced Shortcuts for Professionals
------------------------------------

### Alignment Shortcuts

*   Align Left: 
    *   **Windows/Mac:** Shift + Alt + L

*   Align Center: 
    *   **Windows/Mac:** Shift + Alt + C

*   Align Right: 
    *   **Windows/Mac:** Shift + Alt + R

### Layer Navigation Shortcuts

*   Move Up Layer: 
    *   **Windows:** Ctrl + Up Arrow
    *   **Mac:** Cmd + Up Arrow

*   Move Down Layer: 
    *   **Windows:** Ctrl + Down Arrow
    *   **Mac:** Cmd + Down Arrow

Creating Custom Shortcuts in Gravit Designer
--------------------------------------------

Gravit Designer allows customization of shortcuts to suit your unique workflow preferences. Here’s how to do it:

1.   Open Preferences: Go to Edit, then select Preferences.
2.   NAVIGATE to Shortcuts: Find the Shortcuts tab.
3.   Customize Shortcuts: Click on the action you want to change, and input your desired shortcut.
