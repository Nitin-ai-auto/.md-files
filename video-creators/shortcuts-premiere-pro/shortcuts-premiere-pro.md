Essential Premiere Pro Shortcuts
--------------------------------

### 1. Basic Editing Shortcuts

*   **Cut / Ripple Delete:**
    *   Windows: Ctrl + K
    *   Mac: Command + K

*   **Undo / Redo:**
    *   Windows: Ctrl + Z / Ctrl + Shift + Z
    *   Mac: Command + Z / Command + Shift + Z

*   **Copy and Paste:**
    *   Windows: Ctrl + C / Ctrl + V
    *   Mac: Command + C / Command + V

### 2. Navigating the Timeline

*   **Go to Previous / Next Edit Point:**
    *   Windows: Shift + Left Arrow / Shift + Right Arrow
    *   Mac: Shift + Left Arrow / Shift + Right Arrow

*   **Zoom In / Out on Timeline:**
    *   Windows: = / -
    *   Mac: = / -

### 3. Tool Shortcuts

*   **Selection Tool:**
    *   Windows/Mac: V

*   **Razor Tool:**
    *   Windows/Mac: C

*   **Track Select Tool:**
    *   Windows/Mac: A

### 4. Playback and Preview Shortcuts

*   **Play / Stop:**
    *   Windows/Mac: Spacebar

*   **Step Backward / Forward:**
    *   Windows: Left Arrow / Right Arrow
    *   Mac: Left Arrow / Right Arrow

### 5. Exporting Shortcuts

*   **Export Media:**
    *   Windows: Ctrl + M
    *   Mac: Command + M

Customizing Your Shortcuts
--------------------------

Adobe Premiere Pro allows you to customize shortcuts to suit your editing style. This feature is particularly useful for user-specific workflows or if you're used to shortcuts from other video editing software.

*   To Customize: 
    1.   Navigate to "Edit" > "Keyboard Shortcuts."
    2.   Use the search bar to find the shortcut you want to change.
    3.   Click on the current shortcut and input your desired key combination.

Premiere Pro shortcuts are keystrokes or key combinations that allow you to execute commands quickly without using the mouse.
