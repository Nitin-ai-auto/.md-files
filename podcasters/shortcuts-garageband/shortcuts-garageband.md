Essential GarageBand Shortcuts
------------------------------

### Basic Mac Keyboard Shortcuts

*   New Project: Command (⌘) + N
*   Open Project: Command (⌘) + O
*   Save Project: Command (⌘) + S
*   Undo: Command (⌘) + Z
*   Redo: Shift + Command (⌘) + Z

These shortcuts are foundational and will help you manage your projects from the outset.

### Edit Functions

*   Cut: Command (⌘) + X
*   Copy: Command (⌘) + C
*   Paste: Command (⌘) + V
*   Delete: Delete Key

These shortcuts are essential for managing audio clips, making your editing processes quicker.

### Navigational Shortcuts

*   Zoom In: Command (⌘) + Right Arrow
*   Zoom Out: Command (⌘) + Left Arrow
*   Go to Start: Return/Enter
*   Go to End: Command (⌘) + Right Arrow

These shortcuts aid in navigating through projects quickly, helping you to focus on pick points in your timeline without delay.

### Track and Mixer Shortcuts

*   Show/Hide Mixer: Command (⌘) + Option (⌥) + M
*   Create New Track: Command (⌘) + Option (⌥) + A

These shortcuts enhance your management of components within GarageBand, allowing you to organize your workspace more effectively.

Advanced Shortcuts for Experts
------------------------------

### Automation Commands

*   Show Automation: A
*   Enable Automation: Command (⌘) + Option (⌥) + S

Automation allows you to control volume levels and effects throughout your track seamlessly; learning to automate can transform your projects.

### MIDI and Effects Shortcuts

*   Record MIDI: Command (⌘) + R
*   Toggle Record On/Off: Command (⌘) + Period

These shortcuts allow for quick recording and effects toggling, ideal for intermediate users looking to enhance their tracks.
