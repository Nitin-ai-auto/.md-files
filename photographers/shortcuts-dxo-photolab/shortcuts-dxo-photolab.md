Getting Started with DXO PhotoLab Shortcuts
-------------------------------------------

Before diving into specific shortcuts, it’s essential to familiarize yourself with the different modes and preferences in DXO PhotoLab. The software offers different editing panels, including:

*   Customize Panel
*   Edit Panel
*   Library Panel

### Basic Keyboard Shortcuts

Here’s a list of essential keyboard shortcuts for DXO PhotoLab to get you started:

*   **Open/Import Photos:** Windows Shortcut: Ctrl + O, Mac Shortcut: Cmd + O
*   **Switch to the Library Pane:** Windows Shortcut: Ctrl + L, Mac Shortcut: Cmd + L
*   **Access the Customization Panel:** Windows Shortcut: Ctrl + E, Mac Shortcut: Cmd + E
*   **Zoom In:** Windows Shortcut: Ctrl + +, Mac Shortcut: Cmd + +
*   **Zoom Out:** Windows Shortcut: Ctrl + -, Mac Shortcut: Cmd + -
*   **Reset Adjustment:** Windows Shortcut: Ctrl + R, Mac Shortcut: Cmd + R
*   **Show/Hide Toolboxes:** Windows Shortcut: Ctrl + T, Mac Shortcut: Cmd + T
*   **Export Photos:** Windows Shortcut: Ctrl + Shift + E, Mac Shortcut: Cmd + Shift + E

Advanced Shortcuts for Professionals
------------------------------------

For those accustomed to a more complex editing workflow, these advanced shortcuts can provide additional efficiency:

*   **Quick Selection Tool:** Press _S_ to activate the selection tool in the editing panel.
*   **Toggle Before/After View:** Press _Y_ to compare before and after images seamlessly.
*   **Toggle Color Picker:** Press _I_ to quickly access the color picker for precise adjustments.
*   **Toggle Filmstrip:** Press _F_ to quickly show or hide the filmstrip at the bottom.

Customizing Your Shortcuts
--------------------------

Did you know you can personalize your shortcuts in DXO PhotoLab? This can be particularly useful if you are used to specific commands in other software. To customize shortcuts:

1.   Go to Preferences
2.   Select the Shortcuts tab
3.   Modify existing shortcuts or create new ones tailored to your workflow preferences.
